title: Documents
description: File storage and document management for your organisation.
---

The Documents module provides a centralised place to store and retrieve important files — contracts, permits, certificates, reports, and more.

## Uploading documents

Upload any file type. Each document record stores:

- File name and type
- Upload date and uploader
- Category / tag (optional)
- Description (optional)

Files are stored in Supabase Storage and accessed via secure, organisation-scoped URLs.

## Organisation

Documents can be filtered and searched by name, category, date, or uploader.

## Access control

Only authenticated members of the organisation can access documents. Download links are scoped to authenticated sessions — direct URLs are not publicly accessible.
