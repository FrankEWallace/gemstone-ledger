title: Inventory
description: Stock tracking, write-offs, low-stock alerts, and usage analytics.
---

The Inventory module tracks all gemstone and material stock across your organisation.

## Stock list

Each inventory item records:

- **Name** and category
- **Current quantity** and unit of measure
- **Low-stock threshold** — triggers an alert when quantity falls below this value
- **Cost per unit** and total value

## Quick-adjust

Use the quick-adjust control on any item row to increment or decrement stock directly from the list without opening a full edit form.

## Write-offs

Record stock that is lost, damaged, or discarded via the write-off flow. Each write-off requires a reason and captures who logged it and when.

## Usage analytics

The analytics panel shows consumption over time per item, linked to production log entries that drew from that stock. Use this to forecast reorder points.

## Low-stock alerts

When an item's quantity drops below its threshold, an alert is raised on the Dashboard and in the Notifications panel. Configure thresholds per item via [Alert Rules](/features/alerts).
