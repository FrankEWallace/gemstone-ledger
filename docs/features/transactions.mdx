The Transactions module is the financial ledger for your organisation — recording all income and expenses.

## Adding a transaction

Each transaction records:

- **Type** — Income or Expense
- **Amount** and currency
- **Category** — used for breakdown reporting
- **Date**
- **Customer** (optional, for income entries) — links to a customer profile
- **Notes** (optional)

## Categories

Categories are freeform labels you define. Consistent categorisation drives accurate breakdowns in the [Reports](/features/reports) module.

## Filtering and search

Filter by type, category, date range, or customer. Search by description or amount.

## Reports

The [Reports](/features/reports) module provides income breakdown and expense breakdown views by category, both as charts and exportable PDFs.
