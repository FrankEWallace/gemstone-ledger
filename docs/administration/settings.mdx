## Organisation settings

Owners can configure organisation-level settings including:

- **Organisation name** — displayed throughout the app and in reports
- **Site name** — if your organisation operates multiple sites
- **Logo** — used in PDF report headers

## Profile settings

Each user can update their own:

- **Display name**
- **Email address** — requires confirmation via email link
- **Password**
- **Theme** — light or dark mode

## Module visibility

Owners and admins can enable or disable modules org-wide from **Settings → System → Module Configuration**. Toggleable modules: Messages, Customers, Reports & Analytics, Team & Scheduling, Supply Chain, Operations, and Campaigns. Disabled modules are hidden from the sidebar for every user in the organisation.

## Notifications

Configure which alert events trigger in-app notifications. Users can opt out of specific notification types from their profile.
