---
title: Audit Log
description: Full action history across the organisation.
---

The Audit Log records every significant action taken within your organisation — who did what, and when.

## What's logged

Every create, update, and delete action across all modules is captured, including:

- Record created / updated / deleted
- Module and record ID
- User who performed the action
- Timestamp
- Before/after values for updates (where applicable)

## Accessing the log

Navigate to **Settings → Audit Log**. Only Owners and Admins can view the audit log.

## Filtering

Filter by:

- **User** — see all actions by a specific team member
- **Module** — filter to a specific area (e.g. Inventory, Safety)
- **Action type** — create, update, delete
- **Date range**

## Retention

Audit log entries are retained indefinitely. They cannot be deleted by any user role, including Owners.
